====================================================
  ToriiTranslate API - beállítási útmutató (.env)
====================================================

Nyisd meg a .env fájlt a manga-image-translator mappájában
(ha nem létezik, hozd létre egy üres .env fájlt mellé).

Add hozzá ezt a sort, és cseréld le a saját API kulcsodra:

  TORII_API_KEY=ide_jön_az_api_kulcsod

Például:
  TORII_API_KEY=sk-torii-abc123xyz...


====================================================
  Két config fájl áll rendelkezésre
====================================================

--- torii.json (FULL PIPELINE) ---
  Az egész képet elküldi a Torii /api/v2/upload végpontjának.
  A Torii csinálja a szövegfelismerést, fordítást, eltávolítást
  és renderelést is – egyetlen API hívással.
  Indítás:
    python -m manga_translator local -i C:\Manga\input -o C:\Manga\output --config C:\Manga\manga-image-translator\config\torii.json --use-gpu

--- torii_mixed.json (VEGYES MÓD) ---
  OCR:        Torii /api/ocr       (szövegfelismerés)
  Inpainting: Torii /api/inpaint   (szöveg eltávolítása)
  Fordítás:   OpenAI GPT           (szöveg fordítása magyarra)
  Indítás:
    python -m manga_translator local -i C:\Manga\input -o C:\Manga\output --config C:\Manga\manga-image-translator\config\torii_mixed.json --use-gpu


====================================================
  torii.json testreszabása (Full pipeline módban)
====================================================

  Elérhető fordítói modellek (torii_model):
    "gemini-3-flash"          <- alapértelmezett, gyors
    "gemini-2.5-flash-lite"
    "deepseek"
    "grok"
    "kimi"
    "gpt-5.1"

  Elérhető betűtípusok (torii_font):
    "wildwords"               <- manga stílus
    "anime-ace"
    ... (lásd a Torii API dokumentációját a teljes listáért)

  Szöveg igazítás (torii_text_align):
    "auto"  /  "left"  /  "center"  /  "right"

  Keret kikapcsolása (torii_stroke_disabled):
    false  <- keret van (alapértelmezett)
    true   <- nincs keret

  Minimális betűméret (torii_min_font_size):
    12   <- alapértelmezett


====================================================
  Hitel (kredit) fogyasztás
====================================================

  /api/v2/upload  (teljes fordítás)  : 1+ kredit/kép
  /api/ocr        (szövegfelismerés) : 1 kredit/kérés
  /api/inpaint    (szöveg törlés)    : 0.02 kredit/kérés
